# Educational-Finance

This repositery is a library of educational notebooks that I design as lecture notes/additional materials for my students :)

## Contents

- **Time Series**: ARIMA models and time series analysis
- **Event Studies**: Difference-in-Differences and event study methodologies
- **Cryptocurrency Research**: Herding behavior analysis and cryptocurrency market dynamics
- **Guest Lectures and Lecture Notes**: Additional educational materials

## Cryptocurrency Research

New section covering cryptocurrency market analysis, including:
- Herding behavior measures (CSSD and CSAD methodologies)
- State-dependent analysis (bull/bear markets, volatility regimes)
- Practical implementation with real data sources
- Comprehensive visualizations and statistical tests

Perfect for students and researchers interested in behavioral finance and cryptocurrency markets.
